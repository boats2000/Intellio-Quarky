# Intellio Quarky Project Hub
A collection of custom programs for the **Intellio Quarky** Rover. This is an independent project hub for sharing logic and documentation.

---

## Prerequisites
* **Software:** PictoBlox 9.1.0 (Windows Alpha Version for Intellio).
* **Mode:** Use **Stage Mode** for keyboard controls.

## Project 1: PC Keyboard Controlled Rover
This program provides full manual control of the Quarky Rover build with a custom steering calibration sequence.

### Program Logic
![Rover Logic Screenshot](rover-logic.png)

### Features
* **Driving:** Up/Down Arrows.
* **Steering:** Left/Right Arrows (Servo S1).
* **Speed:** Spacebar (Cycles 25% increments).
